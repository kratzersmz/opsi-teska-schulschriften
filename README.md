# opsi-teska-schulschriften

Mod um alle Schriften zu importieren, Grundscript von MS

* Schriften in unterordner fonts legen
* das script löscht zuerst alle gefundenen fonts aus dem System. Dies war notwendig, da vorherige Schulschriften-opsi Pakete Fehler bei der Schrifteninstallation produzierten, welche dann in Word sichtbar wurden. Hier waren bei der Schriftauswahl dann leere Zeilen oder Fehler beim Schriftenauswählen.
* danach installiert das Script alle Schriften aus fonts
